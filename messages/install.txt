A6s for Sublime Text
==============================

Thanks for installing A6s.

Prerequisites
-------------
Install and run the A6s CLI daemon:

  brew install autonoma-cli
  a6s code --daemon

The daemon listens on ws://localhost:9876/ws by default. Change the port in
Preferences > Package Settings > A6s > Settings.

Getting started
---------------
- ctrl+alt+a  Invoke agent
- ctrl+alt+e  Explain selection
- ctrl+alt+r  Refactor selection
- ctrl+alt+v  Review selection
- ctrl+alt+t  Generate tests for selection
- ctrl+alt+l  List background tasks

All commands are also available from Tools > A6s and the command palette.
